# アブストラクトゲーム『形四 -Keishi-』 公式ルール Ver.2.0

## 概略
6×6の盤上で4つの石を操り、相手より先に「四角形」を描く陣取りパズル。

## 勝利条件
自分の色の石4つを頂点とする「長方形（または正方形）」を完成させる。

- 条件1： 4点が盤の罫線に対して平行であること。（ひし形は不可）
- 条件2： 辺となる石と石の間には、最低1マスの空きが必要。（2×2の最小正方形は不可）

## 準備

- 盤面： 6×6マス（1～6行 x A～F列）
- 駒： 黒4つ（先手）、白4つ（後手）
- 初期配置： 最初は盤面に何もない状態からスタートする。

## ゲームの流れ
ゲームは「第1幕：布石」と「第2幕：闘石」の2つのフェーズで進行する。

### 第1幕：布石（ふせき）
- じゃんけんで先手・後手を決める。
- 盤面が空の状態から、**先手番から交互に1つずつ**好きなマスに石を置いていく。
- お互いが手持ちの石（4つ）を全て置き終わるまで繰り返す。
- **重要：** この配置の途中であっても、勝利条件（四角形の完成）を満たせばその時点で決着（勝利）とする。

### 第2幕：闘石（とうせき）
- 互いに石を置き終わったら移動フェーズに移る。
- 先手有利を相殺するため、このフェーズは**「後手（白）」から**手番を開始する。
- 交互に自分の石を1つ動かす。動かし方は以下の2種類。

  - 歩（あゆみ）：
    - 縦・横・斜め、任意の方向に1マス進む。
    - 移動先に石がある場合は進めない。
  - 越（こし）：
    - 隣接する石（敵味方問わず）を1つ飛び越え、その直後の空きマスに着地する。
    - 連続ジャンプは不可。飛び越えた石は取らない。

## 特記事項

千日手の禁止（コウ）： 直前の相手の手によって盤面が変わった直後、すぐに自分の石を動かして「1手前と全く同じ盤面」に戻してはならない。
