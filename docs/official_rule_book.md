# アブストラクトゲーム『形四 -Keishi-』 公式ルール Ver.5.0

## 概略
6×6の盤上で4つの石を操り、相手より先に「四角形」を描く陣取りパズル。

## 勝利条件
盤上の自分の色の石4つを頂点として長方形（または正方形）を完成させる。

- 条件1： 4点が盤の罫線に対して平行であること。（ひし形は不可）
- 条件2： 辺となる石と石の間には、最低1マスの空きが必要。（2×2の最小正方形は不可）

## 準備

- 盤面： 6×6マス（1～6行 x A～F列）
- 駒： 黒4つ（先手）、白4つ（後手）
- 初期配置： お互いの石が盤面中央で向かい合って一列に並んだ状態からスタートする。
  - **黒（先手）：** B3, C3, D3, E3 （下から3段目の中央一列）
  - **白（後手）：** B4, C4, D4, E4 （下から4段目の中央一列）

## ゲームの流れ
- **先手（黒）から**交互に自分の石を1つ動かす。
- 手番プレイヤーは、以下の2つのアクションのいずれかを選択する。

  1. **【歩（あゆみ）】**：
     - 縦・横・斜めのいずれかに1マス進む。
     - 移動先に石がある場合は進めない。

  2. **【越（こし）】**：
     - 隣接する石（敵味方問わず）を1つ飛び越え、その直後の空きマスに着地する。
     - 連続ジャンプは不可。飛び越えた石は取らない。

## 特記事項

千日手の禁止（コウ）： 直前の相手の手によって盤面が変わった直後、すぐに自分の石を動かして「1手前と全く同じ盤面」に戻してはならない。
